# HybridQBF_CAQE_V2
This repository includes executables and source code of HybridQBF_CAQE_V2. You can run the program directly or recompile it from source.

1. Run executables directly
(1) Set execute permissions
chmod +x bloqqer caqe hqspre HybridQBF_CAQE_V2 HybridQBF_CAQE_V2_M1 HybridQBF_CAQE_V2_M2

(2) Run the solver
./HybridQBF_CAQE_V2 <input_file>

Example: ./HybridQBF_CAQE_V2 benchmark.qdimacs

2. Recompile from source code (source code is in src folder)
(1) Compile HybridQBF_CAQE_V2_M1
First build: ./compile.sh
Normal build: make

(2) Compile HybridQBF_CAQE_V2_M2
First build: ./compile.sh
Normal build: make

(3) Set execute permissions
chmod +x bloqqer caqe hqspre HybridQBF_CAQE_V2 HybridQBF_CAQE_V2_M1 HybridQBF_CAQE_V2_M2

(4) Run the solver
./HybridQBF_CAQE_V2 <input_file>

Example: ./HybridQBF_CAQE_V2 benchmark.qdimacs
